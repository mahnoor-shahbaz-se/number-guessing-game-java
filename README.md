# Number Guessing Game (Java)

A simple Java console application where the user tries to guess a randomly generated number.

## Overview
This project was built to practice programming fundamentals, logical thinking and problem solving using Java.

The application generates a random number and the user keeps guessing until the correct number is found, with hints provided after each attempt.

## Features
• Random number generation  
• Interactive console-based gameplay  
• Hint system (higher/lower feedback)  
• Input validation  
• Attempt counter  
• Simple and beginner-friendly design  

## Concepts Used
Java fundamentals  
Control structures (if/else, loops)  
Random number generation  
Functions and modular programming  
Input handling  

## How to Run the Project
1. Download or clone the repository  
2. Open the project in IntelliJ IDEA  
3. Run the Java file (`NumberGuessingGame.java`)  
4. Follow the console instructions to play the game
